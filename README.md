unity-mesh-builder
=====================

Primitive mesh builders for Unity.

![Demo](https://raw.githubusercontent.com/mattatz/unity-mesh-builder/master/Captures/Demo.png)

## Usage

```cs
Mesh mesh = PlaneBuilder(
    5f,     // width
    10f,    // height
    2,      // width segments
    4       // height segments
);
```

## Primitives

- Plane (Parametric)
- Cube
- Cylinder
- Sphere

Another primitives are in progress ...
